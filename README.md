## self_trained_bot

### 執行方法
執行dialog/MainDialog.cs後複製獲取到的api網址，在Bot Framework Emulator 利用此api 創建一個bot

### 使用語言及技術
C#、Azure Bot Service、CLU、自然語言處理、雲端服務
